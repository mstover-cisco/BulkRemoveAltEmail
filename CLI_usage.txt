Default values are provided for the column names (primary_email and alternate_email), so you only need to specify them if your CSV headers are different. For example, if your CSV headers are 'main_id' and 'email_to_delete':

python BulkRemoveAltEmail.py --csv-file users_to_process.csv --primary-column "main_id" --alternate-column "email_to_delete"
